# World Crests and Feeling Lines

## Status

Provisional v1.3 world naming and feeling-line set.

Date recorded: 2026-06-27

This file defines the current working world names, temporary/provisional place names, World Crest names, and feeling lines for the 64-page ApexCrest first edition.

The names in this file are **working locks**, not final manuscript locks. They are recorded so the repo stays aligned while world names, place names, visual testing, and chapter prototyping continue.

## Purpose

Each world needs a line that makes the customer instantly understand what kind of card belongs there without using mechanical card-placement language.

The feeling line must create the reaction:

**I know exactly which card belongs here.**

It must also support the stronger reaction:

**A binder does not do this.**

## Core rule

The line may use the word **card** because the emotional history belongs to the card.

The page/system uses **Crest** because the Crest is the symbolic recognition moment.

Correct hierarchy:

- World: **Ember Tribute**
- Place: **Volcanic Valley**
- World Crest: **Ember Crest**
- Feeling line: **For the card that still glows after the moment has passed.**

The Crest name must be married to the world name. It should use the first word, one of the main words, or a clearly related root from the world name plus **Crest**.

Do not replace every use of **card** with **Crest**.

Do not use **card** with mechanical verbs such as insert, store, swap, organise, protect, catalogue, or mount in customer-facing copy.

## Reopen rule

These names may be reopened after:

1. the world-name audit;
2. visual testing;
3. the first complete Ember chapter prototype;
4. any evidence that a name feels weak, confusing, too literal, too abstract, too cheesy, or visually mismatched.

## Guardrails

Do not turn these into prompts.

Do not add fill-in fields beneath them.

Do not use them as instructions to insert cards.

Use them as authored recognition lines inside the book.

## Final rule

Each World Crest must make the customer feel:

**This is where that card belongs.**

Not because the book stores it.

Because the world recognises it.
