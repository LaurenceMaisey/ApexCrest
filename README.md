# ApexCrest Brand System

ApexCrest is a premium illustrated collector-book system for the cards and collectibles that earned their place.

This repository is the source of truth for the ApexCrest brand, product, visual, VAT-safe design, commercial, and launch rules.

## Current strategic lock

ApexCrest is not a binder, album, storage product, or display accessory.

ApexCrest is a premium illustrated collector book built around emotional landscapes, meaningful collecting, and the ritual of Cresting.

## Core phrase

**For the cards that earned their place.**

## Repo sections

- `00_MASTER_CONTROL` — project overview, locked decisions, decision log
- `01_BRAND` — brand foundation, archetypes, language system
- `02_PRODUCT` — book structure, worlds, product architecture
- `03_DESIGN` — visual harmony, illustration rules, page anatomy
- `04_VAT_AND_COMPLIANCE` — VAT-safe constraints, HMRC interpretation, risk register
- `05_COMMERCIAL` — pricing, manufacturing, pre-order validation
- `06_LAUNCH` — launch strategy, creator partnerships, content hooks

## Working rule

If a future idea conflicts with a locked decision, it must be reviewed before it is added to the product system.
